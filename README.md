# homebridge-daikin-AirPurifier
A homebridge plugin providing support for daikin air cleaner(MCK70 Series or ACK70 Series)

## Sample


## Installation

```shell
npm install -g .
```

## Configuration
```json
"accessories": [
    {
        "accessory": "DaikinAirPurifier",
        "name": "Daikin",
        "serial": "8888aaaaAAAA",
        "model": "MCK70W",
        "host": "host"
    },
    ...
]

```
